# Context PDFs for AI Responses

This directory contains PDF files that serve as context for the AI to generate responses in the chat interface. 

## Instructions
1. Place your PDF files in this directory.
2. Use the `extract_text.py` script to extract text from the PDFs.
3. Use the `create_embeddings.py` script to generate embeddings from the extracted text.
4. Use the `index_embeddings.py` script to create an index of the embeddings for efficient searching.
